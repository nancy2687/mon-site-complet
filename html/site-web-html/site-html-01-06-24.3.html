HTML

Le HTML c'est quoi ?

Le html est le langage de programmation utiliser sur la plupart des sites internet.
C'est un langage trés utiliser et très populaire. Il est souvent utiliser
pour confectionner des site internet comme celui que je suis en train de faire actuellement.


Pour vous donnez un exemple, voici le code utiliser pour coder cette page que je vais détailler ensuite.

Pour commencer nous allons détailler la première ligne: sur la première ligne il y a marqué Doctype HTML,
cette phrase est utiliser pour montrer au navigateur( quand le programme est executé) que c'est une page en html.
Toutes page HTML doit commencer par une première ligne ou il est marqué Doctype HTML.

Ensuite sur la deuxième ligne il y a une balise conteneur html. Les balises sont les éléments essentiel d'une page HTML,
Ce sont ce qui permet de définir les paragraphes, les sauts de lignes, les titres, les listes, le texte en gras etc.....
La balise html est une balise conteneur parce que pour l'utiliser il faut la déclarer et la fermer avec un / .

La balise br est un exemple de balise vide parce que il n'y a pas besoin de la fermer, il faut juste la déclarer et elle fait ce que on lui demande.
La balise br est utiliser pour faire un sauts de ligne et permettre au paragraphe de ne pas être sur une seule ligne.
