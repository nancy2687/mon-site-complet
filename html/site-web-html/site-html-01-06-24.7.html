Le C++

C++ est aussi un langage de programmation mais contrairement à Python qui est un langage de Haut niveau, c'est-à-dire que il est plus proche du langage humain que de la machine; le C++ est un langage de bas niveau il est donc plus proche de la machine que de l'humain.

Le C++ est moins connu que python parce que on ne l'apprend pas au lycée étant donnée que le programme est à 80% basée sur Python.
